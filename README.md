# Mais Vidros Salvador - SEO Local & UX Premium 💎

Este é um projeto de alta performance desenvolvido em **Astro 4.x**, focado em dominar o mercado de vidraçaria em Salvador através de SEO Técnico, Copywriting persuasivo e Design de Agência.

## 🚀 Status: Pronto para Produção

O projeto foi construído seguindo rigorosos padrões de UI/UX e otimização para mecanismos de busca (SEO Local).

## 🛠️ Tecnologias Utilizadas

- **Framework:** [Astro](https://astro.build/) (Static Site Generation / SSR)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **SEO:** Schema.org (LocalBusiness JSON-LD), Sitemap automático, Meta Tags dinâmicas.
- **Content:** Astro Content Collections (Markdown/YAML)
- **Deployment:** Preparado para [Vercel](https://vercel.com/) com adaptador oficial.
- **Design:** Foco em "Respiro Visual" (py-32) e Alta Conversão (p-12).

## 📈 Recursos Implementados

1. **Central de Configuração:** Dados NAP (Name, Address, Phone) centralizados em `src/data/config.ts`.
2. **SEO Local:** Integração nativa com Google Maps através de Schemas estruturados.
3. **UX Conversiva:** Botão flutuante de WhatsApp com animação de pulsação e formulários otimizados.
4. **Performance:** Código limpo, carregamento de fontes otimizado e zero dependências desnecessárias.
5. **Mobile First:** Menu mobile indestrutível com bloqueio de scroll e animações suaves.

## 📦 Como rodar o projeto

```bash
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev

# Gerar build de produção
npm run build
```
